# NPC Gen2 — EP 8 · เงื่อนไขความสำคัญของ Flip (SRF) + ครึ่งแท่ง 50% + Mother Bar

> Lesson note by Norms (Max) · 2026-05-21 · sources: ATS primary + Turbo

## TL;DR

- **EP 8 = เงื่อนไขความสำคัญของ Flip (SRF)** — ขยายเรื่อง SRF จาก EP 2 ที่พูดไว้แบบน้ำจิ้ม วันนี้ลงรายละเอียดว่าเราจะ "ใช้" Flip ตัวไหน เมื่อไหร่ ในเงื่อนไขแบบไหน
- **กฎ 0** ของ SRF: แบ่งพื้นที่ Buy/Sell ด้วย Fibo 50% ของ Swing แล้ว "ใส่ Order Flow" ลงไป — ของหลักเอาจาก HTF (1H) ก่อนเสมอ
- **5 เงื่อนไข Flip** (1) Flip นอก BSC ต้องเด้งแล้วไปเลย (2) Flip ที่ 50% ของ HTF ต้องมี Reaction (3) Flip นอก Flow = ไม่เทรด มีไว้เป็น Lick (4) ควรรอ Flip ที่อยู่ใน Flow ของ HTF (5) ใน Flow หา POI ครั้งนอก 1-2 ตัว + ครั้งใน 1-2 ตัว เพื่อ confirm
- **ครึ่งแท่ง (50%)** ของ HTF candle — เนื้อเทียนสำคัญกว่าไส้เทียน (เพราะเนื้อเทียนคือ Liquidity ตัวจริง)
- **Mother Bar** — แท่งใหญ่เต็มเนื้อที่ "เปิด" ทิศทาง · set up ต้องไปตามทิศของ Mother Bar
- **Wyckoff lens** หลิวเฉลย: ทั้ง EP ที่สอนมาคือ Wyckoff แบบเล่าผ่านเรื่อง S/R · การเด้งจาก Flip นอก = SOS / Reaccumulation · POI = Lick ก่อน → POI หลัก → POI ตัว confirm

## หัวข้อใน EP นี้

1. ทวน EP 1-7 ก่อน (เพราะต่อจาก EP 7 ที่หยุดเรื่อง MTF+LTF Hilo ค้างไว้)
2. แผน Trade ของเช้าวันนี้ — สอนเรื่องครึ่งแท่ง (50%) ของ Day/4H/1H/15M ที่ทับซ้อนกัน
3. เงื่อนไขความสำคัญของ SRF (Flip) — 5 ข้อหลัก
4. Mother Bar — แท่งเปิดที่บังคับทิศทาง set up
5. การใช้ Flip นอก Flow / ใน Flow / ที่ 50% / POI ครั้งนอก ครั้งใน
6. พฤติกรรม 50% — เด้งทันที vs เด้งหลังเลี้ยงตัว
7. Wyckoff reveal: SOS / Reaccumulation / Spring คือสิ่งที่กำลังสอนอยู่
8. Preview EP 9: TP 1,000 จุด + Miss Entry

## เนื้อหาเต็มที่หลิวสอน

### 1) เปิดด้วยแผนเช้านี้ — เรื่อง "ครึ่งแท่ง" (50% of candle)

หลิวเปิดคลาสด้วยการชวนดูแผนเช้าก่อน เพราะแผนเช้ามี "ของน่าหา" ที่จะใช้ในตอนถัดไปของ EP — เป็นตัวเชื่อมเข้าเรื่อง SRF ที่จะสอนเต็มในชั่วโมงถัดไป

**หลักการครึ่งแท่ง (Half-Candle Rule):**
- เวลาราคา "เบรกออกมาด้วยเนื้อเทียนเต็ม ๆ หนึ่งแท่ง" — มันมักจะ "กลับไปได้ครึ่งหนึ่ง" ของแท่งนั้น
- แบ่งจากเนื้อเทียนอย่างเดียวก็ได้ (พื้นที่ระหว่าง Open-Close) หรือแบ่งทั้งแท่ง (รวมไส้ — High to Low) ก็ได้
- ใช้ประเมินว่าเวลาเบรกออกไป มันจะวิ่งย้อนกลับมาเท่าไหร่ — "ครึ่งแท่ง" ง่าย ๆ ไม่ต้องคำนวณซับซ้อน
- ถ้าดูไม่ออกว่าเนื้อหรือไส้ → "อันไหนถึงเร็วสุด เอาอันนั้นก่อน"
- หลักนี้ใช้ได้กับทุก TF: Day, 4H, 1H, 15M — แค่เปลี่ยน scale

**ตัวอย่างเช้านี้ (XAUUSD เช้า ~3300 area):**
- ครึ่งแท่ง Day → ~3290
- 4H ดูแท่งใหญ่ลำบาก เพราะแท่งใหญ่กินพื้นที่กว้าง → แบ่งเป็น **Swing** (Hi-Lo ของรอบล่าสุด) แทน
- 1H ใช้เนื้อเทียน + ไส้เทียน → ครึ่งหนึ่งอยู่แถว ~3295
- 15M ใช้ Fibo ของชุดสวิงล่าสุด → ทับใกล้กัน ~3295
- พอ 1H, 4H, 15M Fibo ทับซ้อนกัน → ตรงนั้นกลายเป็น **POI = "จุดรวมยุทธศาสตร์"** (Confluence)

**Confluence Stack (ที่หลิวสร้างให้ดู):**
- 50% ของ 4H + 50% ของ 1H + Fibo 50% ของ 15M + BSL 15M (Buy-side Liquidity) ทับซ้อนในจุดเดียว
- เมื่อข้อมูลหลาย TF ชี้จุดเดียวกัน = "จุดรวมยุทธศาสตร์" — เป็นจุดเทรดที่น่าสนใจ
- ภาพยังไม่ชัด 100% (เพราะมี Fibo BSL ที่ยังไม่ถูก clear) → ถ้าอยู่ใน LTF อยากเล่นเร็ว → เปลี่ยน POI นั้นเป็น **Pure Lick (PYLICK)** — มันจะต้องโดน clear ก่อน
- หลังโดน clear → ราคา "สไลด์" ขึ้นมา → สร้าง "ฟอร์มสุดท้าย" (Final Setup) ที่คนรอเทรด
- ค่อยยิง Fibo 1.618 ของ swing ใหม่ → รอราคามาแตะ 50% ของ swing นั้น ค่อยเล่น

**Take-away ก่อนเข้า SRF:**
> "ไม่ต้องคิดเยอะ ถ้ามันมาแตะแถว ๆ นี้ อาจจะต้องเล่น เพราะเป็นจุดรวม"

หลิวเตือนว่าการเทรดมี 2 mode:
1. **Aggressive (รีบเข้า)**: เข้าทันทีที่ POI โดน touch → SL ใหญ่
2. **Patient (รอ confirm)**: รอ Lick โดน clear ก่อน → SL เล็ก แต่ entry ช้ากว่า

### 2) ทวน EP 1-7 (เพราะต้อง bridge เข้า EP 8)

หลิวอธิบายว่าทำไมต้องทวน — เพราะ "การปูพื้นความเข้าใจแต่ละคนใช้เวลาไม่เท่ากัน" บางคนหัวไว จำสูตรแบบกล่อง/เงื่อนไขสั้น ๆ ก็พอ แต่บางคน (รวมตัวหลิวเอง) "สงสัยไปเรื่อย" — เลยต้องเล่าซ้ำ ๆ จนกว่ามันจะซึม

| EP | เนื้อหา |
|----|---------|
| EP 1 | แท่งเทียน + High/Low + S/R (Support & Resistance) |
| EP 2 | SRF (Support–Resistance–Flip) แบบเกริ่น น้ำจิ้ม — ยังเป็นเรื่อง Candlestick ในหมวด SRF |
| EP 3 | Dow Theory — Uptrend/Downtrend/Sideway + นิยาม |
| EP 4 | Sideway (ทีเด็ดของหลิว) — "เจอ Sideway เหมือนเจอเงิน" |
| EP 5 | Dow Theory เชิง Liquidity — ตลาดสะท้อนทุกอย่างไว้หมดแล้ว |
| EP 6 | Multiple Time Frame + Multiple Structure (เริ่มต้น) |
| EP 7 | High/Low ร่วมกันระหว่าง MTF กับ STF (ของผสมเพิ่ม: POI + Session) |

**ประเด็นสำคัญที่ทวน:**
- "Up" สนใจ High ใหม่ ไม่ได้สนใจ Low — ถ้ายังไม่หลุด Low มีโอกาสขึ้นต่อ
- เราชอบ "ให้อภัย" Low ที่หลุดนิดหน่อย (ถ้าสุดท้ายมันทำ Higher High ใหม่ได้)
- หลิวไม่ชอบคนที่เถียงว่า "ที่ไหนดีกว่า" — ถ้าเทรดง่ายขนาด pattern 1:1 จริง คนทั้งโลกรวยหมดแล้ว
- ตลาดให้ข้อมูล 2 ฝั่งตลอด (bullish vs bearish) — เราต้องอ่านทั้งสองด้านพร้อมกัน

### 3) เงื่อนไขความสำคัญของ SRF (Flip) — หัวใจของ EP 8

หลิวเปิดว่า "เงื่อนไข SRF" ไม่ได้พิเศษอะไรขนาดนั้น แต่เป็น "เครื่องมือดูทิศทาง" ของหลิว

**กฎ 0 (สูตรแม่):**
> **แบ่งพื้นที่ Buy/Sell ด้วย Fibo 50% ของ Swing → ใส่ Order Flow ลงไป**

- เหนือ 50% = พื้นที่ SELL
- ต่ำกว่า 50% = พื้นที่ BUY
- (หลิวยอมรับว่าไม่แน่ใจคนแรกที่พูดเรื่องนี้ — อาจเป็น ICT หรือคนก่อนหน้า แต่ "เอาตามโลกปัจจุบัน" ก็ได้)

#### 3.1 เลือก Order Flow ตัวไหน เมื่อมีหลายตัว

ปัญหาที่ทุกคนเจอ: ในกราฟมี Flow เล็กในใหญ่ ทับซ้อนกันหลายตัว → ใช้ตัวไหน?

**คำตอบของหลิว:**
- **เลือกอัน "ใหญ่/หลัก" จาก High Time Frame ก่อนเสมอ**
- ของเล็กข้างใน → ผลักไปเป็น **POI** แทน (ไม่ใช่ Flow หลัก)
- ถ้าอยากเทรดเร็วใน LTF → ปรับ time frame ลง 1/3 ถึง 1/4 (1H → 30M หรือ 15M) แล้วใช้ Flow ของ LTF เพื่อเข้า

**ทำไมต้องเอาจาก HTF?**
> "ถือ order มา มั่นใจที่สุดตอนกดออเดอร์ ตอนกดเสร็จเริ่มเต๋อหมด … จุดที่จะออกจากภาวะกดดันได้เร็วที่สุด คือวัดที่เนื้อเทียนใน Time Frame ใหญ่"

#### 3.2 5 เงื่อนไข Flip (SRF)

หลิวจัดระบบให้ใหม่สำหรับ Gen 2 (ต่างจาก Gen 1 นิดหน่อย):

**Rule 1: Flip ที่อยู่ "นอก" BSC (Buy/Sell Zone) — เด้งแล้วต้องไปเลย**
- ถ้าใช้ Flip ที่ลอยอยู่นอกพื้นที่ Buy/Sell — ต้อง **เด้งทันที** ไม่ใช่ค่อย ๆ เลี้ยง
- ส่วนใหญ่มันไม่เด้งทันที — มันชอบเด้งหลังกลับตัวรุนแรง เช่น Wyckoff Reaccumulation หรือ Wyckoff Spring ใน TF ใหญ่

**Rule 2: Flip ที่ 50% ของ HTF — ต้องมี Reaction**
- ต้องมี "ตบ/กระตุก/เด้ง" (Reaction) ใด ๆ ก่อน
- ไม่ใช่ว่ามาแตะแล้วทะลุไปเฉย ๆ
- ถ้ามีแล็คชั่น = OK, buy/sell ได้ (SL ตามความเสี่ยงรับได้)

**Rule 3: Flip ที่อยู่ "นอก" Flow — ไม่เทรด แต่มีไว้เป็น Lick**
- คือ "ชอบ ๆ ขัดสายตา" ที่เห็นในกราฟ แต่ใช้เทรดไม่ได้
- มีไว้สร้าง Liquidity / เป็น Lick ให้ราคาวิ่งไปกิน

**Rule 4: ควรเลือกเทรด Flip ที่อยู่ "ใน" Flow ของ HTF**
- คือ Flip ที่อยู่ในกรอบ Order Flow หลัก
- มันจะมาแปะกันใน POI ของ HTF

**Rule 5: ใน Flow ของ HTF หา POI ครั้งนอก 1-2 ตัว + ครั้งใน 1-2 ตัว**
- **5.1 ครั้งนอก (MTF)**: เอาไว้ confirm พฤติกรรมราคา + ใช้ตัวที่ "ถึงก่อน"
- **5.2 ครั้งใน (POI หลัก)**: ราคาควรเริ่มกลับตัวแถวนี้ ไม่ค่อยเกิดการเบรกแล้ว

> เนื้อหาตรงนี้หลิวบอกว่า "ใบเบิ้ลเลย" — เป็นโครงคิดในการวางออเดอร์

#### 3.3 POI ครั้งนอก vs ครั้งใน — กลไก

- **POI ครั้งนอก = Lick** (จุดล่อ — คนเข้าใจว่าหลบ มือใหม่/มือกลางมาเทรดตรงนี้)
- **POI ครั้งใน = จุดดูดคน** (คนเริ่มเข้าหนัวกัน แข่งกัน "อันทิปลาย" ดีไม่ดี)
- **POI ตัวที่ 3 (สุดท้าย)** = ตัว Reverse จริง — รอตัวนี้

หลิวเทียบ ICT: "Turtle Soup" ของ ICT = Spring/UTAD ของ Wyckoff → แล้วเขาเทรด "Major Soul ย้อนเป็น Breakout Retest ใน Time Frame เล็ก" — มันคือเรื่องเดียวกัน ที่หลิวสอน คือ "รอ confirm vs ไม่รอ confirm" เท่านั้นเอง

### 4) Mother Bar — แท่งเปิดที่บังคับทิศ

> "แต่ก่อนผมเรียกมันว่า 'แท่งเปิด' … หลังเพิ่งเข้าใจว่าเขาเรียก Mother Bar"

**นิยาม:**
- แท่งใหญ่เต็มเนื้อตัน ๆ ที่ "เปิด" ทิศทาง (Range / Trend ใหม่)
- เป็นแท่งที่กิน range ของแท่งก่อนหน้าหลายแท่ง — engulfing แบบหนักมือ
- มี Momentum ชัด + Volume ชัด (ถ้าดู Volume ด้วย)

**กฎการใช้:**
- เวลาวาง set up → ให้ไปตามทิศของ Mother Bar เท่านั้น
- ห้ามทำ set up ฝั่งตรงข้าม ถึงแม้จะเห็น Lick เห็น POI สวยแค่ไหน
- รอให้มี Mother Bar ก่อน — ถ้ายังไม่มี = ยังไม่เปิดเกม

**ประวัติของหลิว:**
- หลิวคุยกับน้องชายมานานแล้วว่า "รอได้รอ — รอให้มี 'แท่งเปิด' ก่อน"
- ตอนยังไม่รู้จักชื่อ Mother Bar ก็ใช้แนวคิดนี้อยู่แล้ว
- พอเจอชื่อ Mother Bar ก็ตลกดี — "เออ ก็จันหาตั้งชื่อ ตามที่ไอ้นี่นะครับ"

**ความเชื่อมโยงกับ Wyckoff / SMC:**
- Mother Bar = Order Block (OB) ในภาษา SMC = แท่งสุดท้ายก่อนเบรก
- กึ่งกลาง Mother Bar = 50% Equilibrium = SSL/BSL implicit
- ขอบบน/ล่าง = POI ของ retracement
- หลังเกิด Mother Bar → ราคามักย้อนมาทดสอบ 50% (Refinement) → ถ้าผ่าน → ไปต่อ

**Pattern จดจำ:**
- "สายตาของทุกคนหลอกกันไม่ได้" — เห็น Mother Bar ปุ๊บ ทุกคนเห็นตรงกัน → คนจะมารอเทรดที่ POI ขอบ
- กลายเป็น Self-fulfilling Prophecy → ทำให้ Mother Bar ทำงานได้บ่อย
- แต่อย่าลืม: ถ้าเห็นชัดเกินไป (เจาะ POI 2-3 ครั้งแล้วยังไม่ไป) → อาจเป็น Liquidity grab → ระวัง

### 5) Flow ของ LTF ใช้ได้ — แต่ต้องวางเงื่อนไข HTF ก่อน

เคสที่ HTF (1H) มองไม่เห็นโครงสร้างเลย → ปรับให้ 15M เป็น **Main Structure** ได้

**กฎ:**
- ต้องวางเงื่อนไข HTF ให้ครบก่อน (กำหนด trend, bias)
- พอ 15M เป็นโครงสร้าง → 15M สามารถ "โดน clear ได้" เป็นปกติ (เพราะกฎหลักอ้างอิง HTF)
- LIQUIDITY จาก HTF ยังคงอยู่ — อย่าลืมตามไปด้วยเมื่อเปลี่ยน TF

> "ต่อให้โครงสร้างมันดูว่าตรงนี้ไม่ควรเบรก — แต่จงเข้าใจว่ามันเบรกได้ เพราะเราแค่ปรับมาใช้โครงสร้าง TF เล็กเพื่อให้รีเทสต์ มันโดนเบรกได้ แต่ยังคงกลับตัวขึ้นได้"

### 6) พฤติกรรม 50% — เด้งทันที vs เด้งหลังเลี้ยงตัว

นี่คือ "เกล็ดความรู้" ของ 50% ที่ทวนจาก EP เก่า:

**Pattern A — มาแรง ๆ ทันที (โอกาสเด้งครั้งเดียวต่ำ):**
- ราคาวิ่งกระชากขึ้นมาแรง ๆ ปังเข้ามาหา 50% เลย
- ตรงนี้ "ไม่ใช่จุดจบ" — ยังมีอีกรอบ
- ถ้าเป็นขาลง: จะขึ้นไปอีกแล้วค่อยลง

**Pattern B — เลี้ยงตัวก่อน (โอกาสเด้งครั้งเดียวสูง):**
- ราคาเลี้ยงตัวเองสักระยะ → ค่อย ๆ ชน 50% → เด้งลงได้เลย
- 0-1-2-3-4-5 (5 = 50%): มากระชาก → เด้งครั้งเดียวยาก
- 0-1-2-3-4 ค่อย ๆ ขึ้นจนถึง 5: เด้งครั้งเดียวง่ายกว่า

**เหตุผลกลไก:**
- เนื้อเทียน HTF อยู่เหนือ 50% เกือบเสมอ
- ราคามักวิ่งมาหา "เนื้อเทียน" ก่อน ไม่ใช่หยุดที่ "ไส้เทียน"
- ดังนั้น 50% = Flip ระดับเดียวกับเนื้อเทียนที่เบรกออกไป

**Reaction เป้าหมาย:**
- ใกล้ ๆ 50% (49, 48, 47 หรือ 51, 52, 53) เด้งได้ — บวก/ลบ 1-3% ไม่ซีเรียส
- การเล่น 50% ควรเล่น "เต็มโซน" (เพราะ SL ไกล) → ทำเป็น base order
- ปิดทยอย: ปิดกับ Flip/POI ย่อย ๆ ก่อน เผื่อมันไม่ไปต่อ

### 7) Wyckoff Reveal — สิ่งที่กำลังสอนมาคือ Wyckoff ทั้งหมด

นี่คือ "เคล็ดวิชา" ที่หลิวเปิดในครึ่งหลังของ EP — เป็นช่วงที่นักเรียนหลายคนเริ่ม "เห็นภาพ" ว่าทำไมหลิวสอนเรื่อง S/R ซ้ำ ๆ มาตลอด

> "ผมสอน Wyckoff อยู่ตลอดเวลา — ผมแค่สอนผ่านการเล่าเรื่องง่าย ๆ ที่เรียกว่าแนวรับ–แนวต้าน … ถ้าผมเรียก Wyckoff ทุกคนจะงงและไม่อยากจะเรียน"

**ทำไมต้องอ้อม?**
- Wyckoff ในตำราเดิมใช้ศัพท์เยอะ (PS, SC, AR, ST, BU, LPS, JOC, JAC, SOS, etc.)
- คนส่วนใหญ่เห็นศัพท์เยอะ → ปิดหน้าไม่อ่านต่อ
- หลิวเลยสอนแบบ "Stealth Wyckoff" — เรียก S/R, Flip, POI, Lick ทับลงไป
- พอนักเรียนเข้าใจ "เรื่อง" แล้ว → ค่อยเปิดศัพท์ Wyckoff → mental model ผูกกันได้

**Mapping:**
| สิ่งที่หลิวสอน | คือ Wyckoff |
|---|---|
| Flip นอก + เด้งไปเลย | **SOS** (Sign of Strength) — ตำแหน่งลอยที่ขึ้นไปได้ |
| การเด้งนอก Flow | **Reaccumulation** |
| ก้อนสะสมก่อนเบรก | **Accumulation** (Phase A-D) |
| Mother Bar + Spring | **Spring** (Phase C) |
| 50% Reaction | **Test** (Phase B/D) |
| POI ครั้งนอก | **Lick** ที่กิน Liquidity |
| POI ครั้งใน confirm | **LPS** / **JOC** (Jump Across the Creek) |

> "ทุกครั้งที่จะเกิดการเบรก จะเกิดการสะสมก่อน" — นี่คือหัวใจ Wyckoff

**คำเตือน:**
- Wyckoff ไม่ใช่เรื่องตามหาได้ง่าย ๆ → ถ้าตั้งใจหาจะไม่เจอ
- ต้องฝึก "จับพฤติกรรม" ให้เห็นภาพ
- ฝึกตี S/R, Trend Line, Time Line ให้เยอะ แล้วดูพฤติกรรมตอนกลับตัว

### 8) Practical Workflow — Setup ใน 1 หน้าจอ

หลิวสรุปท้ายชั่วโมงด้วยตัวอย่างเทรดจริง XAUUSD (Non-farm session + CPI session):

**Sequence:**
1. ดู HTF (1H/4H) → ตี Swing → Fibo 50% → แบ่ง Buy/Sell Zone
2. หา **Flip นอก Flow** เพื่อทำเป็น Lick (1-2 ตัว) — รอ clear
3. หา **POI ใน Flow** (1-2 ตัว) — confirm
4. ดูที่ 50% ของ HTF — มี Reaction ไหม
5. ถ้า set up ครบ → วาง Pending order
   - ถอดสมอง: เข้าทันทีที่ POI ครั้งใน (SL ใหญ่)
   - รอหน่อย: รอ clear ครั้งนอกก่อนแล้วเข้า (SL เล็ก)
6. TP: ติด Lick / โซน Liquidity ถัดไป (1.618 Fibo extension)

**Bias Management:**
- ระวัง "Chord ค้านสายตา" — มันลงมาเหมือนเสียโครงสร้าง แต่กลับขึ้นได้
- ดูภาพไกลก่อน (HTF) — ขาขึ้น? ขาลง? Sideway?
- ถ้าภาพไกลเป็นขาขึ้น แต่ลงมาจบที่ Low → equal high → BSL → เด้งสั้น ๆ แล้วขึ้นต่อ

**News Event Discipline (Non-farm/CPI):**
- หลิวสาธิตเคส Non-farm: order บน TP ที่ 1:8 (Risk:Reward) เพราะวางตำแหน่งดี
- order ที่อยู่ในกรอบสะสมที่ Non-farm พอออก → ไม่ TP เพราะ momentum ลงต่อ
- กฎ: ออเดอร์ "ใหม่" หลังข่าวจะมี risk สูง → ใช้ 1 ไม้เล็ก ๆ
- ไม่ pending ก่อนข่าวถ้าเห็นไม่ชัด — รอ clear ก่อนแล้วค่อยเข้า

**POI ติด Flow = Liquidity:**
- ถ้า POI อยู่ติด Flow Line → "เท่ากับ Liquidity" → ราคามักจะเคลียร์
- ใช้เทรดยาก ตัวเดียวไม่พอ
- ต้องหา POI Lick + POI confirm มาเสริม

**ใจร้อน vs ใจเย็น (จากตัวอย่างจริง):**
- ใจร้อน: เทรดด้วย Flow → ได้ entry เร็ว แต่ SL อาจโดน
- ใจเย็น: รอ clear ครั้งนอกก่อน → entry ช้า แต่ "ส่วนใหญ่ตลาดวิ่งทาง bias ทำสำเร็จ" → ได้กำไรชัดกว่า

### Wyckoff/SMC Theory เชื่อมโยง

**Wyckoff Phase Mapping (สำหรับ Accumulation):**
- **Phase A**: PS (Preliminary Support) → SC (Selling Climax) → AR (Automatic Rally) → ST (Secondary Test)
- **Phase B**: การสะสม — Building Cause
- **Phase C**: Spring / Test of Spring — false breakdown
- **Phase D**: SOS (Sign of Strength) + LPS (Last Point of Support) — ขึ้นในกรอบ
- **Phase E**: JOC (Jump Across the Creek) — ออกจากกรอบ Markup

**สิ่งที่หลิวเรียก = Wyckoff:**
- "Mother Bar" → Spring candle ที่ Phase C
- "Flip นอก Flow เด้งเลย" → SOS bar ใน Phase D
- "POI ครั้งใน" → LPS (Last Point of Support)
- "POI ครั้งนอก" → Lick = Liquidity Sweep ที่ Phase B/C

**SMC Concepts ที่ Cross-reference:**
- **BOS (Break of Structure)** = เบรก Higher High/Lower Low → Bias confirm
- **CHOCH (Change of Character)** = เบรก swing แรกของขาตรงข้าม → trend change
- **OB (Order Block)** = แท่งสุดท้ายก่อนเบรก = "Mother Bar" ของหลิว
- **FVG (Fair Value Gap)** = generic gap — หลิวไม่ใช้ ICT-FVG เพราะ NPC ใช้ Wyckoff+SMC เท่านั้น
- **Liquidity** = generic Lick (BSL/SSL) — รวมกับ POI ของหลิว

### Fibonacci ในระบบ NES

- **0.5 (50%)**: แบ่ง Buy/Sell zone — เป็น "ใจกลาง" ของ Swing
- **0.618**: Premium/Discount boundary (ใช้น้อยใน NPC — หลิวเน้น 50% + 1.618)
- **1.618 Extension**: TP target / "จบชุด Swing" — เนื้อเทียนต้องทะลวง 1.618 ถึงจะนับว่าจบ
- **Miss Entry**: ถ้าราคาแค่มาแตะ 1.618 (ไม่ปิดเนื้อเทียนผ่าน) = "ยังเป็น Miss Entry" → EP 9 จะต่อเรื่องนี้

### Trader Psychology

**กับดักความรู้สึก (ที่ EP 8 ขยี้):**
1. **"เห็นในสิ่งที่อยากเห็น"** — เรียน SMC มาเยอะ → เห็น OB เห็น FVG เห็น Lick ทุกที่ → เทรดทุกที่ → สต็อกบวม → ขาดทุน
2. **"กลัวตกรถ"** — เห็นราคาขึ้นไปแล้วรีบเข้า → กลายเป็นโดนกินไส้เทียน
3. **"ค้านสายตากรรมการ"** — เห็นกราฟเสียโครงสร้างแล้วรีบเซลล์ → กราฟกลับขึ้นต่อ (เพราะภาพไกลยังเป็นขาขึ้น)
4. **"หลอนตลอดทาง"** — กราฟวิ่งระหว่าง High/Low → "เป็น High หรือยังวะ?" → ลังเล → พลาดทั้งสองทาง

**Discipline:**
- มั่นใจที่สุดตอน "กดออเดอร์" — หลังกด ลืมแผนหมด
- → วางเงื่อนไขให้แน่นใน HTF ก่อน เพื่อให้สมองออกจาก stress ได้เร็ว
- ไม่ต้องเทรดทุกครั้ง — "อดทนรอ ได้อันที่ดีที่สุด ยิ่งดี"
- "อย่าเอาตัวเองไปไถกับไฟ" — ถ้าเห็นว่ากำลังเล่นกับ noise → หยุด

**Backtest แบบสมเหตุสมผล (ที่หลิวอ้างถึงวันเสาร์):**
- พอร์ต $1,000 / วาง 5% / 1 อาทิตย์ ชนะ ~6-7 ไม้ / ไม้นึงได้ 1-2 เท่าของเงินลง
- จบอาทิตย์บวก ~60% — ถือว่าเยอะมาก
- อย่าหลงกับคน "วันเดียวโต 500-1000%" — ไม่ยั่งยืน + อาจไม่ใช่เทรดจริง

**เรื่องไม่ได้คุมเกม:**
- หลิวย้ำว่า "เราไม่ได้คุมเกม"
- "ต่อให้แม่นขนาด 99.9% อีก 1% ที่ถืออยู่คือ Owner — คุณก็แพ้ได้แทนที"
- ใช้ Dow Theory เป็น Liquidity → ดึงสติออกจาก "ม่านความสับสน" ของตลาด
- ยอมรับ: เราชนะตลาดในระยะยาวได้แค่ "ถ้าทำสิ่งที่สมเหตุสมผลซ้ำ ๆ"

**Cognitive bias ที่หลิวเตือน:**
1. **Confirmation bias** — เห็นในสิ่งที่อยากเห็น
2. **Sunk cost fallacy** — "เซลล์ผิดแล้ว ไม่ถอดใจ — เซลล์เพิ่ม → ห่อแตก"
3. **Recency bias** — เห็น 1-2 รอบที่ pattern X ใช้ได้ → เชื่อว่า X = สูตรสำเร็จ
4. **Loss aversion** — เลื่อน SL หนีเรื่อย ๆ → loss ใหญ่ขึ้น
5. **FOMO** — กลัวตกรถ → เข้าผิดจังหวะ → กลายเป็น exit liquidity ของคนอื่น

- **EP ก่อนหน้า (EP 7)**: HiLo ร่วมกันระหว่าง MTF กับ STF + POI + Session — EP 8 ต่อยอดด้วยการบอกว่า "เลือก Flip ตัวไหนใน MTF/HTF"
- **EP ถัดไป (EP 9)**: ทำไม TP 1,000 จุด + Miss Entry — EP 8 เปรยไว้แล้วว่า "ปิดเนื้อเทียนผ่าน 1.618 = จบ Swing, แค่แตะ = Miss Entry"
- **อ้างย้อน EP 2**: SRF น้ำจิ้ม — EP 8 คือเวอร์ชันเต็ม
- **อ้างย้อน EP 4**: Sideway "เจอเหมือนเจอเงิน" — ก้อนสะสมก่อนเบรกที่ EP 8 พูด คือ Sideway เล็กใน LTF
- **อ้างย้อน EP 6**: Multiple Time Frame / Multiple Structure — EP 8 ใช้ทักษะนี้ในการเลือก Flow และ POI
- **อ้างไป Wyckoff**: หลิวเฉลยใน EP นี้ว่าทุกอย่างที่สอน = Wyckoff ผ่านแว่น S/R เพื่อให้เรียนง่าย
