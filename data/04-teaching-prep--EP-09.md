# NPC Gen2 — EP 9 · ที่มาที่ไปของ TP 1,000 · ปิดพาร์ทแรก

> Lesson note by Norms (Max) · 2026-05-21 · sources: Turbo transcript

## TL;DR

- **TP 1,000 จุด = ไม่ใช่ตัวเลขมั่ว** มันเป็นค่า "ความเป็นไปได้สูงสุดต่อ swing" ที่คำนวณจาก **กรอบสะสม (Accumulation range) 500–1,000 จุด** ขยายด้วย **Fibonacci 1.618** ออกไปจาก swing ที่เพิ่งจบ
- **เกมของหลิวคือเทรดจากในสะสม → ออกไป TP นอกสะสม** ฉะนั้น "หา TP" = "หาว่ากรอบสะสมอยู่ไหน + ปลายทาง Fibo 1.272/1.618 อยู่ไหน"
- ถ้าเข้าได้ที่ **50% ของ swing (Fibo 0.5)** → TP 1,000 ก่อนถึง 1.618 = **เซฟ และเป็นไปได้แทบทุกวัน**
- **ห้ามเพิ่ม lot เพื่อหวังกำไรเยอะ** เพราะ leverage = ตัวคูณที่หด SL จริงให้สั้นลง พอร์ตจะหายเร็วโดยที่ไม่รู้ตัว — อยากได้เยอะกว่า ต้อง **ฝึกเข้าให้ลึก (ปลายๆ swing)** ไม่ใช่อัดหลอด
- EP นี้ปิดพาร์ทแรก (Intro + พื้นฐาน) — พาร์ทสอง = Liquidity, สัปดาห์หน้าเริ่ม

## 🎯 หัวข้อใน EP นี้

1. ทำไมหลิวสอนช้า — ปรัชญา "ต่อจิ๊กซอว์" และเหตุผลที่ไม่สอน Setup ตรงๆ
2. ความจริงเรื่อง Leverage — ทำไมรวยจาก Levelage ยั่งยืนไม่ได้
3. **หัวใจของ EP 9** — TP 1,000 มาจากไหน? ที่มาทางคณิตศาสตร์ของ Fibonacci 0.5 + 1.618
4. ทำไมต้อง "เทรดจากในสะสม TP นอกสะสม" — เกมที่เราเล่นอยู่
5. TP Buy = Sell Limit — ทำไม Liquidity ที่ 1.618 ถึงทำงาน
6. การประเมิน "ความเป็นไปได้สูงสุด" ของ Swing ก่อนเข้าออเดอร์
7. กรณีพิเศษ — ปี 2026 กรอบสะสมขยายเป็น 2,000–4,000 จุด, ต้อง scale ใหม่
8. Recap พาร์ทแรก (EP 1–9) — โครงสร้างเนื้อหาที่เรียนมาแล้ว
9. Live walkthrough — ใช้กราฟจริงจัดทำแผน + รัน backtest หน้างาน

## 📚 เนื้อหาเต็มที่หลิวสอน

### 1) ปรัชญาการสอน — ทำไมต้องช้า

หลิวเปิด EP 9 ด้วยการ "เคลียร์ใจ" ก่อน เพราะมีนักเรียนทักมาบ่นว่า **"เมื่อไหร่จะได้เทรดสักที"** — หลิวบอกตรงๆ ว่า:

- ที่ผ่านมา (EP 1–8) คือ **"บทนำ + ปรับ Mindset"** ทั้งหมด ไม่ใช่ขั้นสอน
- ตอนสอน Gen 1 ตั้งใจให้จบ "ขั้นต้น" ที่ EP 3 ด้วยซ้ำ แต่ Gen 2 ลากยาว เพราะหลิวประเมินว่า **ทุกคนมาด้วย "สภาพ" ที่ต่างกัน** บางคนใจร้อน บางคนทุนน้อย บางคนไม่มีรายได้ประจำ → หลิวต้อง "เป็นเบรกให้ทุกคน"
- **ภาพเปรียบเทียบ "จิ๊กซอว์"** ที่หลิวใช้ตลอด:
  - หลิว = คนต่อจิ๊กซอว์
  - หน้าที่นักเรียน = "ยกชิ้นที่ตัวเองมีอยู่แล้ว ขึ้นมาวางบนโต๊ะให้ครบ"
  - แล้วเดี๋ยวหลิวต่อให้ดูที่ละชิ้นว่ามันเสียบเข้าตรงไหน
- **ห้าม "กลับไปสู่ลูปเดิม"** — ลูปเดิม = เห็น Setup สวยๆ แล้วเทรดทันที, เห็นข้อมูลที่ "ตลาดอยากให้เห็น" แล้วเข้าเทรด — เพราะเราไม่ใช่คนต่อภาพ เราคือ "คนรอเก็บช่องโหว่"

> **คีย์ลึก:** ถ้าหลิวสอน Setup สั้นๆ ให้ตอนนี้ คนรู้สึก "คุ้มค่าคอร์ส" ทันที — แต่จะกลับไปแพ้แบบเดิม เพราะลูปความคิดยังเดิม

หลิวประกาศตรงนี้ว่า — **"จะไม่เตือนเรื่องวิธีการอีกแล้ว"** ใครใจร้อน เทรดเอง รับผิดชอบเอง หลิวจะไปต่อตามแผนเดิม "ช้าๆ + ทวนตอนแรก"

### 2) ความจริงเรื่อง Leverage — ที่ไม่มีใครพูด

หลิวพูดตรงประเด็นเลย:

> **"คุณไม่สามารถรวยจาก Leverage แบบยั่งยืนได้ — Leverage ถูกออกแบบมาเพื่อให้เงินในกระเป๋าออกเร็วขึ้น"**

ภาพเปรียบเทียบ:
- คุณมี 5 บาท อยากซื้อทองคำ 50 บาท → **ความจริงคือคุณจะเสีย 5 บาทไปเรื่อยๆ จนเสีย 200–2,000**
- โลกนี้ไม่แฟร์พอที่คนมีเงิน 5 บาท จะไปซื้อทองคำ 50 บาทได้ — ความเป็นจริงเป็นแบบนี้
- Leverage = ลูกเล่น เอามาทำให้คุณ "เสียเร็วขึ้น" ไม่ใช่ "รวยเร็วขึ้น"

หลิวจะ "ไม่ห้าม" ใช้ Leverage ต่อเมื่อ:
1. เรียนจบแล้ว (พาร์ท 1 + 2 + 3)
2. ปั๊กกำไรนิ่งมาอย่างน้อย **4–5 เดือน**

ก่อนหน้านั้น = **ฝึกที่ 0.01 lot อย่างเดียว** เพื่อให้ "ชินตัวเลข" — เพราะปัญหาจริงๆ ของเทรดเดอร์คือ "กลัวตัวเลข" ไม่ใช่กลัวกราฟ

### 3) หัวใจ EP 9 — TP 1,000 มาจากไหน?

#### กลไกการเกิด TP 1,000

หลิวอธิบายผ่านนิทาน **"หลวงพ่อบู๊ / ลาบูบู๊"** เพื่อให้เห็นภาพ:

> สินค้าเริ่มต้นที่ราคา 10 บาท → ถูกโปรโมต → เป็นกระแส → ราคาวิ่งขึ้นทำ High → คนเริ่มเก็งกำไร → จนถึงวันที่ "ความนิยมลด" → ราคาเริ่มพักตัว → คนเทขายเพราะเงินไม่พอ → เกิดการสะสมราคารอบใหม่

**คำถามคือ:** ราคาจะหยุดที่เท่าไหร่? จะวิ่งไปไหน?

**คำตอบ:** ใช้ **Fibonacci** เพราะเป็น "เครื่องมือสากลที่สุดในโลก" — ทุกคนทั้งโลกใช้ตัวเดียวกัน เลยเกิด **Liquidity จริง** ที่จุด 0.5, 1.272, 1.618

#### คณิตศาสตร์ของ TP 1,000

ภาพหลักที่ต้องวาดในหัว — **Fibonacci บนกรอบสะสมขนาด 500–1,000 จุด**:

```
ราคา (แกนซ้าย)
1,618  ───── TP สูงสุด (1.618 extension)
1,272  ───── TP คนขี้กลัว
1,000  ───── High (100%)
  786  ───── จุดเข้าที่ดี (Fibo 0.786 retracement)
  618  ───── จุดเข้าปลอดภัย (Fibo 0.618)
  500  ───── ครึ่งทาง (Fibo 0.5) ← เข้าซื้อ "หลับตา"
  300  ───── (Fibo 0.382)
  220  ───── (Fibo 0.236)
    0  ───── Low (0%)
```

**ตารางเข้าซื้อ vs TP 1,000:**

| จุดเข้า | TP ที่ 1.272 | TP ที่ 1.618 | TP 1,000 ได้ก่อนไหม? |
|---------|-------------|-------------|---------------------|
| 500 (Fibo 0.5) | +772 จุด | +1,118 จุด | **ได้แน่ก่อน 1.618** |
| 380 (Fibo 0.382) | +890 จุด | +1,238 จุด | ได้ก่อน 1.618 |
| 300 (Fibo ~0.3) | +972 จุด | +1,318 จุด | ได้ก่อน 1.618 (เซฟกว่า) |
| 220 (Fibo 0.236) | +1,052 จุด | +1,398 จุด | ได้ก่อน 1.272 ด้วยซ้ำ |

> **คีย์สำคัญ:** ยิ่งเข้าได้ลึก (ใกล้ Low ของ swing) → ยิ่งได้ TP 1,000 ก่อน "ระยะเป้าของคนอื่น" (1.618) → **เราออกก่อนชาวบ้าน เรารอด**

> **คีย์อีกชั้น:** ถ้าเข้าได้ที่ 50% และ TP 1,000 — ความเสี่ยงต่ำมาก เพราะ 1,000 จุด **มาก่อน 1,118 (1.618)** ความเป็นไปได้สูงสุด → ไม่ต้องลุ้นว่าจะถึง 1.618 จริงไหม

#### ทำไม 1.618 ถึงทำงาน — กลไก Liquidity

- **1.618 = Golden Ratio** ทุกคนทั้งโลกที่ใช้ Fibo จะวาง TP ไว้ที่นี่
- **TP Buy = Sell Limit** (เพราะการปิด Buy = การขาย)
- ดังนั้นที่ 1.618 จะมี **Sell Limit เต็มไปหมด** → กลายเป็น **Liquidity Pool** → ราคาวิ่งไปแตะแล้วมักจะกลับ
- เราต้อง **TP ก่อน 1.618 นิดนึง** เพราะ:
  - ไม่รู้ว่าจะมี Sell Limit มากแค่ไหน
  - บางที Sell Limit ถูกกินหมด → ราคาทะลุไปต่อ (กลายเป็น Sell เข้าตลาด ราคาลง)
  - บางที Sell Limit เยอะ → ราคาเด้งกลับ ไม่ถึง 1.618

### 4) ทำไมต้องเทรด "จากในสะสม → TP นอกสะสม"

หลิวย้ำคีย์ที่พูดมาตลอด EP 1–8 อีกรอบ:

**"หัวใจการเทรดของหลิว = รอการสะสม (Accumulation)"**

- เจอกรอบสะสม = **เจอเงิน** (เพราะเป็น "ไซส์เว" หรือ Sideway ที่มีเงินรอ)
- เหตุผล: เพราะกรอบสะสม = พื้นที่ที่ Smart Money กำลัง absorb supply/demand เพื่อจะเบรกออกในที่สุด
- การเทรดจากในสะสมออกไป = ขี่คลื่นที่ตลาดสร้างขึ้นเอง ไม่ฝืน

**ทำไมห้ามเทรดข้างนอกสะสม (เทรด Breakout):**

- เทรด Breakout = "เห็นเขาสร้างร้านมาร่าเต็มถนนแล้ว ค่อยมาเปิดร้าน" — สาย!
- คุณจะไม่กล้าขาย เพราะกำไรไม่พอ
- คุณจะ chase ราคา → SL กว้าง → RR แย่
- Smart Money ใช้ Breakout เป็น **กับดัก** เพื่อกินคนเข้าสาย

> **เปรียบเทียบ:** สมัยร้านมาร่าสายพานบูม (2022–2023) — เพื่อนชวนเปิดร้าน หลิวบอก "กูไม่เปิด ต้นทุนหายแน่นอน" → ในตลาด ถ้าเห็นกระแสแล้วค่อยเข้า = เจอ Distribution ของจริง

### 5) เครื่องมือที่ใช้จริง — Fibo 50% + 1.618

> หลิวพูดชัด: "เครื่องมือเรามี 2 ค่าเท่านั้น — **Fibo 0.5 กับ Fibo 1.618**"

**วิธีลาก Fibo:**

1. หา **Swing หลัก** (High ↔ Low ของก้อนล่าสุด) — ใช้ "ราคาปิด" หรือ "เนื้อเทียน" เป็นหลัก
2. ลาก Fibo จาก Low → High (ถ้าจะเข้า Buy)
3. ดู:
   - **0.5** = จุดเข้า "หลับตา" (เซฟกลางๆ)
   - **0.618 / 0.786** = จุดเข้าปลายๆ (เสี่ยงน้อย กำไรเยอะ)
   - **1.272** = TP คนขี้กลัว
   - **1.618** = TP เต็ม Swing (Golden Ratio)
4. ถ้าจะลากไส้เทียนด้วย ก็ทำได้ — เพื่อดู "ขอบเขตทั้งหมดที่เป็นไปได้"

**กฎเซฟ:**

- ลาก Swing ใหม่ทุกครั้งที่มี High/Low ใหม่
- ถ้า 1.618 ไปไม่ถึง = ราคามีอะไรอยู่ตรงทาง (Flip, POI, Liquidity) → ประเมินใหม่
- ถ้า 1.618 ทะลุไป = อาจมี Swing ใหญ่กว่าซ่อนอยู่ → ดู Time Frame ใหญ่ขึ้น

### 6) การประเมิน "ความเป็นไปได้สูงสุด" ก่อนเทรด

ก่อนกดออเดอร์ทุกครั้ง หลิวคิด 3 ชั้น:

**ชั้นที่ 1 — Scale ของกรอบสะสม:**
- ปกติ = 500–1,000 จุด
- ปี 2025–2026 (XAUUSD) = ขยายเป็น 1,000–4,000 จุด → ต้อง scale ใหม่

**ชั้นที่ 2 — ระยะที่เป็นไปได้สูงสุด:**
- ลาก Fibo 1.618 จาก Swing หลัก → นี่คือ "เพดาน"
- ถ้าเข้าที่ 50% → กำไรสูงสุด = ครึ่งหนึ่งของ Swing + extension

**ชั้นที่ 3 — Risk vs Reward:**
- ถ้า RR < 1:1 หรือ < 1:2 → **ไม่เทรด**
- ถ้า RR > 1:3 → ทำตามแผน
- ถ้าอยากได้เยอะกว่า 1,000 จุด → **ห้ามเพิ่ม lot** ให้ **เข้าให้ลึกกว่าเดิม** (ปลายๆ Swing)

> **คีย์:** "ความสามารถสูงสุดที่กราฟวิ่งให้ได้" = สิ่งสำคัญ — ห้ามเดาเกิน ห้าม "กราฟเข้าข้างตัวเอง"

### 7) กรณีพิเศษ — ปี 2026 กรอบสะสมใหญ่ขึ้น

หลิวอธิบายว่า "ตลาดทุกวันนี้ไม่ปกติ":

- เมื่อก่อน Range สะสม = 500–1,000 จุด → TP 1,000 พอดี
- 2026 (XAUUSD) — Range สะสม = 1,000–4,000 จุด → TP อาจเป็น 2,000–4,000 จุด

**วิธี adapt:**

- ใช้หลักเดิม แต่ scale ตามกรอบจริง
- ถ้ากรอบสะสม 2,000 จุด → TP ที่ 1.618 อาจเป็น 3,236 จุด
- **ห้าม fix ที่ 1,000 ตายตัว** — ต้องดูกรอบสะสมก่อนแล้วค่อยคำนวณ
- ถ้าอยากเซฟแบบเดิม → เข้าให้ลึก, ใช้ TP เล็กลง (TP 1.272 แทน 1.618)

### 8) Live Walkthrough — ใช้กราฟจริง

ในช่วงค่ำหลิวเปิดกราฟ XAUUSD ตั้งแต่ต้นปี 2022 ทำตัวอย่างยาวๆ:

**ขั้นที่ 1 — ตั้ง Time Frame Stack**
- Day = ภาพรวมเทรนด์
- 4H = วางแผน Bias
- 1H = หา Structure หลัก
- 15M / 5M = หา POI / Entry
- 1M = ดูพฤติกรรมการกลับตัว (Confirmation)

**ขั้นที่ 2 — ระบุโครงสร้าง (Wyckoff/SMC mix)**
- หา HH / HL (Up-trend) หรือ LH / LL (Down-trend)
- หาจุด **BOS** (Break of Structure) และ **CHOCH** (Change of Character)
- ถ้านับ Elliott 1–2–3–4 ได้ ก็นับ (แต่ไม่ซีเรียส) — "นับแบบคนไม่เป็น"

**ขั้นที่ 3 — ระบุ POI / Flip / Liquidity**
- POI ติด Flow = **Liquidity** (อันตราย ราคาน่าจะทะลุ)
- POI ลึกเข้า Flow = **จุด Entry จริง**
- ก่อน Break ต้องเกิด **Accumulation** เสมอ — ถ้าไม่มี Accumulation = ไม่เทรด

**ขั้นที่ 4 — ทดสอบ Plan**
- วาง Sell ที่ POI บน + Buy ที่ POI ล่าง (ทั้งสองทาง)
- ทางไหนที่ราคาเดินก่อน = ทำงาน
- TP ที่ Fibo 1.272 หรือ 1.618 ของ Swing

**บทเรียนสำคัญจาก Live:**
- หลิวเองยัง "ตกรถ" บางออเดอร์ (เพราะงก-20 จุด)
- หลิวยอมรับว่า "ผมเอาจุดอ่อนตัวเองมาสอน" — สอนให้ "ไม่พลาดแบบหลิว"

## 🌐 ข้อมูลสากลเสริม

### Wyckoff/SMC Theory เชื่อมโยง

**1) TP 1,000 = หลักการ "Profit Taking at Markup Phase"**

- ใน Wyckoff — หลังจาก Phase D (Markup) → ราคาเข้า Phase E (Distribution หรือ Re-accumulation)
- จุดที่หลิวเรียก "1.618" = ใกล้กับ **PSY (Preliminary Supply)** ในโมเดล Wyckoff Distribution
- การ TP ที่ 1.272–1.618 = ขายก่อน Smart Money จะเริ่ม Distribute → ปลอดภัยกว่ารอ BC (Buying Climax)

**2) Accumulation Range 500–1,000 จุด**

- Wyckoff คลาสสิก: Phase A → B → C → D → E เกิดในกรอบที่ค่อนข้างชัด
- Phase B = "Building the Cause" — ระยะนี้ยิ่งยาว/ใหญ่ → Markup ยิ่งไกล
- 500–1,000 จุด ใน XAUUSD = ระยะที่พอดีให้เกิด Cause + Effect ที่วัดได้ผ่าน Fibo extension

**3) TP Buy = Sell Limit + Liquidity**

- ในศัพท์ Wyckoff: บริเวณ 1.618 = "Resistance ที่ Smart Money วาง Supply Order" (Composite Operator)
- ในศัพท์ SMC: บริเวณ 1.618 = **Buy-Side Liquidity Pool** (ที่ TP Buy ของ retail mass จับกลุ่มกัน) — Smart Money มาเก็บเพื่อ Reverse
- **คีย์:** ทำไมต้อง TP ก่อน 1.618 → เพราะถ้า Buy-Side Liquidity ถูก absorb หมด ราคาจะ reverse แรง

**4) Golden Ratio ในธรรมชาติและตลาด**

- Fibonacci sequence ปรากฏในธรรมชาติ (เปลือกหอย, ดอกทานตะวัน, กาแล็กซี่)
- ในตลาด — ไม่ใช่ "เวทย์มนต์" แต่เป็น **Self-fulfilling prophecy** เพราะคนทั้งโลกใช้ตัวเดียวกัน
- 1.618 / 0.618 = ค่าที่ "เกิด Liquidity ได้จริง" เพราะมีคำสั่งวางอยู่จริง

### Trader Psychology

**1) "อยากเทรดยากๆ หรือ?"**

หลิวถามคำถามนี้ตลอด EP — เป็นคำถามทางจิตวิทยาสำคัญ:
- เทรดฉลาด = เลือกวันง่าย, เลือก Setup สวย, ปล่อยวันยากผ่านไป
- เทรดไม่ฉลาด = เห็นทุกวัน ทุกแท่งเทียน คือโอกาส → เครียด → แพ้

**2) Recency Bias + Confirmation Bias**

- เราเห็นในสิ่งที่อยากเห็น (Buy เพราะอยาก Buy)
- ตลาดรู้ว่าเราอยาก Buy → ตลาดสร้าง Pattern ที่ฟ้องว่า "Buy ได้" → เรา Buy → ตลาด Reverse
- **ทางแก้:** มองหา Accumulation (ที่เราไม่อยากเห็น) แทนที่จะมอง Direction (ที่เราอยากเห็น)

**3) Loss Aversion + Over-leverage**

- คนใจร้อนจะเพิ่ม Lot เพื่อ "เอาคืนเร็ว" → SL หดสั้นลง → แพ้เร็วขึ้น
- หลิวสร้าง "ความกลัวให้ตัวเอง" → ไม่กลับไปบ้าบินอีก
- **กฎ:** ฝึก 0.01 lot ให้ชินก่อน — เปลี่ยน lot ทีหลังเมื่อใจนิ่งจริง

**4) "ขายไม่เป็น" — ปัญหาใหญ่ของ retail**

> "ทุกคนเข้ามาในตลาดพร้อมจะ Buy — แต่ไม่มีใครรู้จัก Sell"

- TP 1,000 = วิธี "บังคับให้ขายเป็น"
- ตัดอารมณ์ ตัดความหวัง — ทำตามตัวเลข
- ขายที่ Plan ไม่ใช่ที่ "รู้สึก"

## 📖 คำศัพท์ที่ควรรู้

| คำ | ความหมาย |
|----|----------|
| **TP (Take Profit / Target Price)** | จุดที่เราตั้งใจปิดออเดอร์เพื่อทำกำไร — หลิวเรียก "Target Price" ก็ได้ |
| **Fibonacci Retracement (Fibo 0.5, 0.618, 0.786)** | ค่าย้อนกลับใน Swing — ใช้หาจุดเข้า |
| **Fibonacci Extension (Fibo 1.272, 1.618)** | ค่าขยายเลย Swing — ใช้หา TP |
| **Golden Ratio (1.618)** | สัดส่วนทอง — ใช้ทั้งในธรรมชาติและตลาด, เป็น "ตัวเลขสากล" |
| **Accumulation (กรอบสะสม)** | พื้นที่ที่ Smart Money absorb supply/demand — Wyckoff Phase B |
| **Distribution** | พื้นที่ที่ Smart Money ขายออก — Wyckoff Phase E |
| **POI (Point of Interest)** | จุดที่ราคามักกลับตัว — ในศัพท์หลิว = Demand/Supply zone, Flip, Order Block |
| **Flip / SRF (Support-Resistance Flip)** | จุดที่ Support เปลี่ยนเป็น Resistance (หรือกลับกัน) |
| **Flow (Order Flow)** | ทิศทาง momentum หลักของราคา |
| **Liquidity** | เงิน/คำสั่ง pending ที่รออยู่ — Smart Money มากินเสมอ |
| **POI LIQ (PY Liquidity)** | คำเฉพาะของหลิว — Liquidity ใน Previous swing |
| **BSL / SSL** | Buy-Side Liquidity / Sell-Side Liquidity (generic SMC) |
| **Equal High / Equal Low** | High หรือ Low ที่ใกล้เคียงกัน 2 ครั้ง — สร้าง Liquidity |
| **BOS (Break of Structure)** | ราคาทะลุ High/Low เดิม — confirm trend |
| **CHOCH (Change of Character)** | ราคาเปลี่ยนทิศ — จาก Up เป็น Down หรือกลับกัน |
| **Spring (Wyckoff)** | False breakdown ใต้ Trading Range → ดูดของ → ราคาขึ้นต่อ |
| **UTAD (Upthrust After Distribution)** | False breakout เหนือ Trading Range → ขายของ → ราคาลงต่อ |
| **Leverage / Levelage** | ตัวคูณ — เพิ่ม lot ขนาด, ลด SL จริงที่รับได้ |
| **RR (Risk:Reward Ratio)** | อัตราส่วนเสี่ยง:ผลตอบแทน — ต้อง > 1:2 ขั้นต่ำ |
| **Time Frame Stack** | การวิเคราะห์หลายไทม์เฟรมพร้อมกัน (Day / 4H / 1H / 15M / 5M) |
| **Focus** | คำเฉพาะของหลิว — POI ตัวที่ "ต้องเทรด" (Confidence สูงสุด) |
| **0.01 lot** | ขนาด lot เล็กที่สุด — ใช้ฝึก, 1 จุด = $0.01 ต่อ XAUUSD |
