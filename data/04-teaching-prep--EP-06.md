# NPC Gen2 — EP 6 · Top Down Analysis · Multiple Structure + Multiple Time Frame

> Lesson note by Norms (Max) · 2026-05-21 · sources: ATS primary + Turbo

## TL;DR

- EP 6 คือบทที่เราเอา "ตลาดมี 3 แนวโน้ม" ของ Dow Theory มาแยกออกเป็น **2 มิติ** — **Structure** กับ **Time** — แล้วเอามาประกอบเป็น **Top Down Analysis** สไตล์ Social Norm System
- เราเลิกจำเพียงแค่ Up / Sideway / Down เพราะมันไม่พอใช้งานจริง · ต้องแยกเป็น **Primary / Secondary / Minor** ทั้งฝั่ง Structure และฝั่ง Time
- หัวใจที่หลิวย้ำตลอด EP คือ "เรากำลังทำอะไรอยู่ตรงไหนของตลาด" — ถ้าตอบไม่ได้ก็จะกลายเป็น **Buy in Sell / Sell in Buy** ซึ่งเป็น loop ตายของคนที่แพ้ซ้ำๆ
- เครื่องมือสำคัญใน EP นี้: 4-step Top Down (TF ใหญ่ → กลาง → เล็ก) · นับ swing 1-2-3-4-5 แบบหยาบๆ (ไม่ใช่ Elliott) · 50% rule · Fibonacci 0.618/1.618 จุดจบ swing · Flow ที่เกิดเมื่อ break HH/LL
- กฎเหล็กจบ EP: **เก่งไม่กลัว · กลัวไม่ได้เก่ง** — ถ้ายังเซฟพอร์ตไม่อยู่ ห้ามวางใหญ่ · เทรดเฉพาะ "ขยักแรกของโครงสร้าง" (ตำแหน่ง 2-3) เท่านั้น

## 🎯 หัวข้อใน EP นี้

- ทวน EP 5 — Dow Theory จบส่วน "ตลาดสะท้อนทุกอย่างไว้หมด" + 3 แนวโน้ม
- Top Down Analysis คืออะไร · ทำไมคนเรียนแล้วยัง confuse
- แยกออกเป็น 2 มิติ: **Multiple Structure** + **Multiple Time Frame** (MTF)
- 3 ระดับของแนวโน้ม: Primary (หลัก) · Secondary (รอง) · Minor (ย่อย)
- Time Frame mapping: Day · 4H · 1H · 15M · 5M · 1M
- หน้าที่ของแต่ละ TF: ใหญ่=Direction+Flow, กลาง=แผน+ข้อมูล, เล็ก=Confirm พฤติกรรม
- การนับ swing แบบหยาบ 1-2-3-4-5 เพื่อประเมิน "จบ/ไม่จบ"
- กฎ 50% และ Fibonacci 0.618/1.618 สำหรับจุดจบ swing
- Flow concept · เมื่อราคา break HH/LL → flow เกิด → จะถูก retest ก่อนวิ่งต่อ
- Buy in Sell / Sell in Buy trap — สาเหตุที่คนแพ้ซ้ำๆ
- Momentum analysis — open/close + DXY + News
- กฎ "เก่งไม่กลัว · กลัวไม่ได้เก่ง" และระบบป้องกันพอร์ตในช่วงฝึก

## 📚 เนื้อหาเต็มที่หลิวสอน

### 1) ทวน EP 5 และจุดเริ่มของ EP 6

จาก EP 1-5 เราปูฐานมาแล้ว — High/Low, Support/Resistance, Candlestick, Dow Theory (Up/Down/Sideway), Liquidity แบบมินิ, และคำกล่าวที่ว่า "ตลาดสะท้อนทุกอย่างไว้หมดแล้ว" จุดสุดท้ายของ Dow ที่เราคุยใน EP 5 คือ "ตลาดมี 3 แนวโน้ม" — Primary (หลัก) · Secondary (รอง) · Minor (ย่อย)

แต่ตรงนี้ถ้าเราเข้าใจแค่ "Up / Down / Sideway" มันยังไม่พอใช้งานจริง — เพราะตลาด ณ ขณะหนึ่งจะมีหลายแนวโน้มซ้อนกันอยู่ ถ้าเราอ่านไม่ออกว่าเรา "อยู่ใน Primary ไหน · ทำอะไรอยู่ใน Secondary · กำลังจะเข้าจังหวะอะไรใน Minor" เราก็จะตกหลุม Buy in Sell หรือ Sell in Buy เสมอ EP 6 เลยมาแก้ตรงนี้

### 2) ปัญหาของ Top Down Analysis ที่คนอื่นสอน

Top Down Analysis ไม่ใช่ของใหม่ ทุกคนน่าจะเคยได้ยินมาแล้วทั้งนั้น แต่เวลาคนสอนกันทั่วๆ ไป เขาจะพูดสองเรื่องผสมกันในประโยคเดียว — เรื่องของ **TF (เวลา)** กับเรื่องของ **โครงสร้าง (Structure)** สลับไปสลับมา จนคนเรียนจับไม่ได้ว่ากำลังคุยอะไรอยู่

ตัวอย่างที่ทุกคนเคยเจอ:
- "ดู TF 1H — เจอ supply zone ใหญ่ — ย่อยลงไป 15M — เจอ refinement — ย่อยลงไป 5M — เทรด"

ฟังดูง่ายตอนเรียน · แต่พอเอาไปใช้จริง แพ้แทบทุกครั้ง · เพราะเรากำลังถูกสอน 2 เรื่องพร้อมกันโดยไม่รู้ตัว ทั้งเรื่อง **กรองโซน (Structure)** และเรื่อง **ย่อยไทม์เฟรม (Time)** — แต่ไม่รู้ว่าเรื่องไหนเป็นหัวใจ

หลิวเลยจับแยกออกเป็น 2 ขั้นชัดๆ ก่อน — **Multiple Time Frame** กับ **Multiple Structure** — เรียนคนละทาง ทำความเข้าใจให้แม่นทีละเรื่อง แล้วค่อยเอามาประกอบกัน

### 3) Multiple Time Frame — มิติของเวลา

หลิวจัด TF เป็น 3 ระดับเทียบกับ 3 แนวโน้มของ Dow:

| ระดับ | ชื่อ | TF ที่ใช้ | ดูอะไร |
|---|---|---|---|
| Primary Time | หลัก | Day / 4H | หลายๆ สัปดาห์ — เทรนด์ใหญ่ + Flow |
| Secondary Time | รอง | 1H / 15M | หลายๆ วัน — วางแผน + วาง POI/zone |
| Minor Time | ย่อย | 5M / 1M | ภายในวัน — confirm พฤติกรรมเข้าเทรด |

**ทำไมเลือกแบบนี้?** เพราะ TF หลัก (Day → 4H → 1H → 15M → 5M → 1M) มันลดลงประมาณ **4 เท่า** ในแต่ละขั้น — เป็นมาตรฐานสากลของทุกแอป ไม่ต้องไปงัดอย่าง 6H / 8H ที่ "เพี้ยน" — ใช้ TF หลักก็พอ

**ข้อยกเว้น:** Day ใช้บ้างเฉพาะตอนกราฟ "หน้าสิ่ว หน้าขวาน" (จะ break ไม่ break) เพื่อหาความเชื่อมั่น · ส่วน 6H/8H ใช้แค่ตอน 4H ดูไม่สวย ปรับลงแล้วเห็นชัดกว่า

#### หน้าที่ของแต่ละ TF (สำคัญมาก)

**TF ใหญ่ (Day/4H) → กำหนด Trend + Flow**
- ดูว่าตอนนี้เป็น Uptrend / Downtrend / Sideway
- กำหนด **Flow** — Flow เกิดเมื่อราคา break HH (ขาขึ้น) หรือ break LL (ขาลง)
- เลือกข้างว่าจะ Buy หรือ Sell — เลือกผิด = ยอมแพ้ง่าย จบวันแพ้ พรุ่งนี้ค่อยใหม่
- กำหนด **จุดยอมแพ้** (SL ภาพใหญ่)

**TF กลาง (1H/15M) → วางแผน + วางข้อมูล**
- เห็นแท่งเทียนชัดขึ้น เห็นรายละเอียดที่ TF ใหญ่ไม่เห็น
- วาง POI (Point of Interest) — order block / demand / supply / liquidity zone / fib
- คำว่า POI หลิวใช้รวม — ไม่แยกถี่ถ้วน เพราะ "มันก็คือ zone เหมือนกัน"
- 15M คือ TF ที่ "พอดี" ที่สุดสำหรับวาง zone — ไม่ใหญ่เกิน ไม่เล็กเกิน

**TF เล็ก (5M/1M) → Confirm พฤติกรรม**
- ใช้เฉพาะ "หน้างาน" เท่านั้น
- รอ confirm พฤติกรรมราคาตามที่วางแผนไว้ (Spring, sweep, BOS เล็กๆ)
- ห้ามเอาไปวางโครงสร้างหลัก เพราะจะเจอ noise + "หลอกตา"

### 4) Multiple Structure — มิติของโครงสร้าง

โครงสร้างแบ่งเป็น 3 ระดับเหมือนกัน:

- **Primary Structure** = โครงสร้างหลัก (ก้อนขนมปังก้อนใหญ่) — กำหนด Direction รวม
- **Secondary Structure** = โครงสร้างรอง (ก้อนข้างใน Primary) — บอกว่า Primary จบ/ยัง
- **Minor Structure** = โครงสร้างย่อย (ขยักเล็กๆ ภายใน Secondary)

**วิธีหา Primary Structure ที่ถูกต้อง:** ขึ้นไป TF ใหญ่ → เลือก HH/LL ที่ "อยู่ไกลที่สุด · เห็นชัดที่สุด" — นั่นคือก้อนหลัก

ที่สำคัญคือ Multiple Structure จะไปด้วยกันกับ Multiple Time Frame เสมอ — แต่ **ไม่ Fix 1:1** กับ TF · บางวัน Primary Structure อยู่ที่ 4H · บางวันอยู่ที่ 1H · ขึ้นอยู่กับโอกาส

### 5) นับ Swing 1-2-3-4-5 (แบบหยาบ — ไม่ใช่ Elliott)

หลิวย้ำชัด: **"ผมนับแบบคนนับไม่เป็น"** — ไม่ใช่ Elliott Wave · ไม่ใช่ A-B-C · เป็นแค่ heuristic เพื่อประเมินว่า swing นั้น "จบหรือยัง"

**กติกาการนับ:**
- เริ่มนับ "0" ที่จุด **Break** (เพราะหลังเบรกต้องมี retest = confirm)
- นับไป 1, 2, 3, 4, 5
- ถ้ามันจะไป 6, 7, 8 ก็ช่างหัวมัน — แค่อยากเห็น 5

**ทำไมต้องนับ 5?** เพราะคนทั่วไปที่นับ A-B-C จะคิดว่ามันจบที่ 3 — แต่ความจริงตลาดจะลากต่อไป 5 — เราเลยเทรดได้ที่ 3 และ 5 (กิน wave คนที่นับ ABC)

#### กฎการเทรดจากการนับ

- เห็น **1 → 2** = รอ 3 (มีโอกาสเกิดสูง)
- เห็น **1 → 2 → 3** = ปลอดภัย เทรดได้
- เห็น **1 → 2 → 3 → 4 → 5** = swing น่าจะจบแล้ว เตรียม reverse
- **เทรดแค่ 2 กับ 3 ก็ปลอดภัยที่สุด** — เพราะถ้า 2 = sell, 3 = buy (ในขาลง)

**Quote หลิว:** "ถ้าเราเทรดแค่ 1-2-3 ก็ปลอดภัย เพราะคนส่วนใหญ่คิดว่าตัวเองถูก มันคิดว่าตัวเองต้องไปให้ถึง TP ... มันก็เลยกลายเป็นว่ามันต้องไปได้ คือเทรดแค่ 3 ก็พอแล้ว"

### 6) กฎ 50% และ Fibonacci

กฎสำคัญที่หลิวสอนปนเข้ามาใน EP นี้ (จะเจาะลึกใน EP ถัดๆ ไป):

**50% Rule:**
- swing ส่วนใหญ่จะ retrace ครึ่งทาง (50%) ก่อนวิ่งต่อตามเทรนด์
- ถ้า price วิ่งเข้า 50% **เร็วเกินไป** = "หลอก" — มันจะย่อหลอกนิดเดียวแล้วขึ้นไปกิน liquidity ก่อนค่อยตามเทรนด์
- ถ้า price **มาจอดที่ 50%** แล้วเด้งหา liquidity ฝั่งตรงข้ามก่อน = "ปกติ" — จะตามเทรนด์
- 50% ไม่ต้องวัด fib ละเอียด — แบ่งครึ่ง swing เอาเลย

**Fibonacci สำหรับจุดจบ swing:**
- **0.618** — โซน rejection ปกติ
- **1.618** — จุดจบ swing โครงสร้างหลัก
- **1.27** — รองลงมา ใช้เป็นเป้าได้
- วัดจาก **ราคาปิดของแท่ง** (ไม่ใช่ไส้เทียน) เพราะ 1.618 จะมาเก็บ liquidity ที่ราคาปิด

### 7) Flow Concept

**Flow คืออะไร?**
- ขาขึ้น: Flow Buy เกิดเมื่อราคา **break HH** — ถ้าไม่หลุด LL ต่อ → มีโอกาสไปต่อ
- ขาลง: Flow Sell เกิดเมื่อราคา **break LL** — ถ้าไม่ break HH → มีโอกาสไปต่อ

**กฎเหล็กของ Flow:**
- ก่อนจะ break ทุกครั้งจะเกิด **การสะสม** (Wyckoff Accumulation/Distribution)
- หลัง break ทุกครั้งจะเกิด **การ retest** — ราคาจะกลับมา "กิน" จุดที่สะสมไว้ก่อน แล้วค่อยวิ่งต่อ
- ตี Flow ที่ **อยู่ใต้ liquidity** เสมอ — ไม่ตี Flow ที่ **อยู่ไกล liquidity** เพราะเสี่ยง
- ตี Flow ที่ผ่าน liquidity ไปแล้ว → ปลอดภัยที่สุด

**Quote หลิว:** "ทุกครั้งที่จะเกิดการเบรก จะเกิดการสะสม สะสมแล้วค่อยทำลาย"

### 8) ลำดับการคิด Top Down 4 ขั้น (ของหลิว)

1. **TF ใหญ่ (4H/1H)** → ดู Trend + Momentum → **เลือกข้าง Buy/Sell ครั้งเดียว**
2. กำหนด **Flow** → ตีโซนใหญ่ๆ ที่ "อยู่ใต้ลิก" หรือ "ผ่านลิกไปแล้ว"
3. **TF กลาง (15M)** → วางข้อมูล/POI → หาพื้นที่เทรดที่อยู่ใน Flow
4. **TF เล็ก (5M/1M)** → Confirm พฤติกรรมราคา → เข้าเทรด

ตัวอย่างจริงจากเมื่อวานในห้องเรียน:
- 4H: เห็น Trend ลง · มีแนวรับยกขึ้น · liquidity zone ข้างบน
- 1H: เห็น Flow ขาลง · ตรงนี้เคยเป็น HL เก่า → กลายเป็น flip
- 15M: หา demand zone / drop-base-rally / order block ที่อยู่ใต้แนวรับ
- 5M: รอ price ลงไปต่ำกว่า PYLIX (liquidity ลิก) + ผ่านเวลา news → เข้า sell

### 9) Buy in Sell / Sell in Buy — ตำนาน Trap

**Quote หลิว:** "เราต้องตีโจทย์ให้ออกว่าเรากำลังทำอะไรอยู่ — Buy ใน Sell หรือ Sell ใน Buy"

ตัวอย่างเมื่อวานในห้องเรียน — ตลาดเปิดมาทุ่มลง 2-3 ครั้ง คนทั่วไปเห็นย่อนิดเดียวก็แห่ Buy เพราะคิดว่า "rejection" — แต่ความจริง:
- TF ใหญ่ = Sell อยู่ (Primary Structure ขาลง)
- ไส้ที่เห็นว่า "rejection" จริงๆ คือการสะสมก่อน break
- คนที่ Buy = Buy in Sell = สวนเทรนด์ = ถ้ารอด ก็ "Buy ไป Sell" สั้นๆ เท่านั้น

**วิธีไม่ตกหลุม:**
1. กำหนด Primary Structure ให้ถูกก่อน (ที่ TF ใหญ่)
2. ถ้า Primary = ขาลง → จะ Buy ก็ได้ แต่ต้อง "Buy เพื่อ Sell" (TP ที่ครึ่งทาง · ไม่ถือยาว)
3. ห้าม Buy ลึก (ถือผ่าน 50% ของ swing)

### 10) "Norm Theory" — Momentum Analysis (เสริมจาก EP 5)

หลิวสอนเสริมเรื่อง Momentum (จะเจาะใน EP ถัดๆ ไป) — โครงสร้างคิดคือ:

**3 ขั้น:**
1. **ตลาดให้ข้อมูล** (open, close, candle pattern, news, DXY)
2. **ผู้คนตอบรับ** (liquidity ที่ถูกวางไว้)
3. **ตลาดจัดการคนเหล่านั้น** (sweep liquidity แล้วไปจริง)

**ตัวอย่างประเมิน Momentum:**
- ดูแท่งปิดวันก่อนหน้า — สีอะไร · อะไรทำให้มันปิดสี่นั้น
- เช็ค DXY — ถ้าข่าวทำ DXY ขึ้น → XAUUSD ควรลง
- ถ้าเปิดวันใหม่สวนกับ close วันก่อน → ระวัง "หลอก" ก่อน sweep แล้วค่อยไปจริง

**กฎ:** ห้ามเทรดด้วยเหตุผลเดียว — ต้องมี confluence อย่างน้อย 2-3 อย่างเสมอ

### 11) "เก่งไม่กลัว · กลัวไม่ได้เก่ง" — Risk Discipline

**Quote หลิว (สำคัญมาก):**
> "เก่งไม่กลัว กลัวไม่ได้เก่ง — คือถ้าระเบิดก่อน จะเอาที่ไหนมาเก่ง"

**กติกาช่วงฝึก (4 เดือนของ NPC Gen 2):**
- เทรดทีละ 2-5% ของพอร์ต · ห้ามเกิน
- แพ้ก็แพ้ — รับว่าแพ้ · พรุ่งนี้เอาใหม่ · ห้ามฝืน
- ห้าม "เติมพอร์ต" หลังแพ้เพื่อเอาคืน (loop ตาย)
- ห้าม "Big Win" mindset — ดูแต่ Big Loss แทน (เพราะ Big Win เห็นน้อย Big Loss เห็นเยอะ)
- ห้ามใจร้อน ใส่ทุกออเดอร์เพราะ "ไม่อยากพลาดโอกาส" — นั่นคือนิสัยพนัน

## 🌐 ข้อมูลสากลเสริม

### Wyckoff/SMC Theory เชื่อมโยง

**Multi-Timeframe Analysis (MTF) ในตำราสากล** เริ่มต้นจาก Charles Dow เอง (1900s) ที่แบ่ง market trend เป็น 3 ระดับ — Primary (1+ ปี), Secondary (3 สัปดาห์ ถึง 3 เดือน), Minor (น้อยกว่า 3 สัปดาห์) — นี่คือ "Three Trend Theory" ที่หลิวต่อยอดมาเป็น Social Norm System โดยปรับเป็น TF ที่ใช้ใน intraday trading ของ XAUUSD/Forex สมัยใหม่

**Wyckoff Phase กับการนับ swing 1-2-3-4-5:** วงจรของ Wyckoff (Accumulation Phase A-E แล้วเข้า markup phase) สอดคล้องกับ swing wave — โดย Phase A (PS+SC+AR+ST) คือ swing 1-2 ของหลิว · Phase B+C (สะสม + Spring) คือ swing 3 · Phase D+E (SOS, LPS, markup) คือ swing 4-5 — แม้ไม่ใช่ mapping 1:1 แต่ logic คล้ายกัน คือ "สะสมก่อน break แล้ว retest แล้ววิ่ง"

**Break of Structure (BOS) ในระบบ SMC** เกิดเมื่อราคา break HH (ขาขึ้น) หรือ LL (ขาลง) — ตรงกับ "Flow เกิด" ของหลิว · CHOCH (Change of Character) คือ break โครงสร้างไปอีกฝั่ง = เปลี่ยน Primary Structure · ระบบของหลิวใช้ logic เดียวกัน แต่ไม่เน้นชื่อทาง technical เพราะอยากให้ผู้เรียนเข้าใจ "ทำไม" ก่อน "ชื่อ"

**Fibonacci 0.618 และ 1.618** มาจาก Fibonacci sequence ที่ใช้ใน Elliott Wave Theory (R.N. Elliott 1930s) — โดย 0.618 = golden ratio retracement (เป็น "deep retracement" ก่อน continuation) และ 1.618 = extension เป้าทาง markup · ในบริบท Wyckoff คือจุดที่ composite operator มักปิด position หรือเริ่ม distribution

### Trader Psychology

**Confirmation Bias** — Trader ส่วนใหญ่เลือกที่จะเห็น setup ที่ตรงกับสิ่งที่ตัวเอง "อยาก" เห็น แล้วมองข้าม signal ตรงข้าม · เหตุนี้คนที่ "นับ ABC ผิด" จะถือต่อไปเพราะคิดว่าตัวเองถูก · กลายเป็นเหยื่อให้คนที่นับ 1-2-3-4-5 (หลิว) กิน · การฝึก "นับให้ขำๆ ไม่ซีเรียส" ของหลิวคือวิธี hedge bias ตัวเอง

**Recency Bias + FOMO** — เห็นกราฟวิ่งลงแรงๆ ก็คิดว่ามันจะลงต่อ (จำได้แค่แท่งล่าสุด) เลย FOMO sell ที่จุดต่ำสุด · แล้วโดนกินที่ retest · นี่คือสาเหตุที่หลิวบังคับให้ "วางแผนก่อนเช้า · เปิด TF ใหญ่ก่อน" — เพื่อ anchor ตัวเองกับ context จริง ไม่ให้ recency มาบงการ

**Loss Aversion (Prospect Theory — Kahneman & Tversky)** — คนกลัวขาดทุนมากกว่าดีใจกำไรเท่ากัน 2 เท่า · ทำให้ trader ที่แพ้แล้วจะ "เติมพอร์ต" เพื่อเอาคืน → loop ของ revenge trade · กฎ "5% per trade · แพ้ก็แพ้ · พรุ่งนี้เอาใหม่" ของหลิวคือ behavioral guardrail ตัด loop นี้ขาด

**Illusion of Control** — Trader ที่เพิ่งชนะหลายไม้จะรู้สึก "เก่ง" แล้วเพิ่ม size · แต่ความจริงคือตลาดให้โอกาสมาเอง ไม่ใช่ skill เพิ่มขึ้น · นี่คือเหตุที่หลิวเตือน "Big Win ไม่ค่อยเห็น เห็นแต่ Big Loss" — เพื่อ check ego

## 📝 การบ้าน

### Practice (ฝึกบนชาร์ต)

1. **MTF Mapping Drill** — เปิด XAUUSD บน TradingView · กดดู Day → 4H → 1H → 15M → 5M ตามลำดับ · ในแต่ละ TF ระบุ Primary Structure (HH/HL หรือ LH/LL) ด้วยตาเปล่า · จด Direction ที่เห็นในแต่ละ TF · ตรวจว่า 3 TF บนสุด **สอดคล้องกัน** หรือไม่ · ถ้าไม่สอดคล้อง → คุณกำลังจะ Buy in Sell / Sell in Buy
2. **Swing Counting Drill** — เลือก 5 swing ขาขึ้นและ 5 swing ขาลงบน chart 1H ย้อนหลัง 1 สัปดาห์ · นับ 1-2-3-4-5 จากจุด break ของแต่ละ swing · สังเกตว่า swing ส่วนใหญ่ "ไปได้ถึง 5" หรือ "จบที่ 3"
3. **50% + Flow Drill** — เลือก 3 swing ที่ break HH/LL ในสัปดาห์ที่ผ่านมา · วัด 50% retracement · วัด Fib 0.618 และ 1.618 · ดูว่าราคาเคารพ levels เหล่านี้ไหม · แต่ละ swing เกิด "การสะสมก่อน break" ที่มองเห็นได้หรือไม่
4. **Top Down Plan Drill** — วันเสาร์-อาทิตย์ทำแผนเทรดสำหรับสัปดาห์ถัดไป · เริ่มจาก 4H → กำหนด Direction + Flow · 1H → กำหนด POI/zone · 15M → วาง entry zone · เขียน scenario A (ตามแผน) + scenario B (ผิดแผน · จุดยอมแพ้)

### Reflection

1. ช่วง 1 สัปดาห์ที่ผ่านมา เทรดที่แพ้ของคุณ — กี่ % คือ "Buy in Sell / Sell in Buy"? (เปิด chart ย้อนดูทีละไม้ · ระบุ Primary Structure ตอนนั้นว่าเป็นฝั่งไหน · ถ้าคุณเทรดสวน = Buy/Sell in)
2. เวลาแพ้ คุณ "เติมพอร์ต / เพิ่ม size / เทรดถี่ขึ้น" เพื่อเอาคืนหรือไม่? · ถ้าใช่ → ลองเขียนกฎตัวเองว่า "ถ้าแพ้ติด 2 ไม้ จะหยุดเทรด 24 ชม."
3. คุณเคยนับ swing แบบไหนมาก่อน (Elliott / ABC / ไม่นับ)? · ลองนับแบบหยาบ 1-2-3-4-5 ของหลิว 1 สัปดาห์ · เทียบดูว่า "เห็นภาพ" ชัดกว่าเดิมไหม

## 📖 คำศัพท์ที่ควรรู้

- **Top Down Analysis** — การวิเคราะห์จาก TF ใหญ่ลงไป TF เล็ก · เพื่อ anchor context ก่อนเข้าเทรด
- **Multiple Time Frame (MTF)** — ใช้ TF หลายระดับพร้อมกัน (Day/4H/1H/15M/5M/1M)
- **Multiple Structure** — ใช้โครงสร้าง 3 ระดับ (Primary/Secondary/Minor) วิเคราะห์พร้อมกัน
- **Primary Structure** — โครงสร้างหลักของ TF ใหญ่ · กำหนด Direction รวม
- **Secondary Structure** — โครงสร้างรอง (ก้อนข้างใน Primary) · บอกว่า Primary จบ/ยัง
- **Minor Structure** — ขยักเล็กๆ ภายใน Secondary · ใช้ confirm พฤติกรรม
- **Flow** — direction ของแรงตลาดที่เกิดเมื่อ break HH/LL · ถ้าไม่หลุดฝั่งตรงข้าม = มีโอกาสต่อ
- **POI (Point of Interest)** — โซนสำคัญ · รวม supply, demand, order block, fib, flip zone
- **Liquidity (Lik)** — กลุ่ม stop loss หรือ pending order ที่ตลาดต้อง "เคลียร์" ก่อนวิ่ง
- **PYLIX (Pure Liquidity)** — liquidity ที่ยังไม่ถูกเคลียร์ · เป้าหมายตลาด
- **Flip** — โซนที่เคยเป็น resistance กลายเป็น support (หรือกลับกัน)
- **Buy in Sell / Sell in Buy** — เทรดสวนกับ Primary Structure · loop ตายของคนที่อ่านโครงสร้างไม่ออก
- **50% Rule** — swing ส่วนใหญ่ retrace ครึ่งทางก่อนวิ่งต่อ · ใช้เป็นเกณฑ์ "จบ/ไม่จบ"
- **Fibonacci 0.618 / 1.618** — golden ratio retracement และ extension · ใช้หาจุดจบ swing
- **Sweep** — การที่ราคาวิ่งทะลุ HH/LL เพื่อเคลียร์ liquidity แล้วกลับ
- **Spring / UTAD** — Wyckoff terms — Spring คือ false breakdown ในขาขึ้น · UTAD คือ false breakout ในขาลง
- **Norm Theory** — กรอบคิด "ตลาดให้ข้อมูล → คนตอบรับ → ตลาดจัดการ" ของหลิว
- **Drop-Base-Rally (DBR)** — รูปแบบ demand zone — ราคาลงแรง · พักนิ่ง · วิ่งขึ้นแรง
